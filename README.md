# 💳 Credit Card Financial Dashboard
> An interactive Power BI dashboard built using transaction 
and customer data from a SQL database to provide real-time 
financial insights.

## Objective
To monitor credit card business performance by analyzing 
revenue, interest earned and transaction patterns across 
card types, customer demographics and spending categories.

## Dataset
- credit_card.csv — transaction level credit card data
- customer.csv — customer demographic details
- SQL database — data extraction and transformation

## Tools Used
- Power BI — dashboard & visualizations
- SQL — data extraction & transformation
- CSV — raw data

## Key Findings
- Blue card holders generate the highest revenue at 47M 
  out of total 57M
- Swipe transactions dominate at 36M compared to 
  Chip 17M and Online 4M
- Graduate customers contribute the highest revenue 
  at 23M by education type
- Businessmen lead revenue by job category at 18M
- Q4 recorded the highest transaction count at 173.2K

## How to View
Download the `.pbix` file and open with Power BI Desktop

## File Structure
- `Credit_Card_Financial_Dashboard.pbix` — Power BI file
- `Credit_Card_Transaction_Dashboard.sql` — SQL queries
- `credit_card.csv` — Transaction dataset
- `customer.csv` — Customer dataset
- `Credit_Card_Report_transaction.pdf` — Transaction report
- `Credit_Card_Report_customer.pdf` — Customer report
