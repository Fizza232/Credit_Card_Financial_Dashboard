## 📌 Project Overview

This project analyzes credit card transaction and customer data to uncover key financial trends and performance metrics. The dashboard is split into two interactive reports — one focused on transactions and one on customer demographics.

### Key Metrics
| Metric | Value |
|---|---|
| Total Revenue | 57M |
| Total Interest Earned | 8M |
| Total Transaction Amount | 45.5M |
| Total Transaction Count | 667.2K |
| Total Customer Income | 588M |
| Customer Satisfaction Score | 3.19 |

---

## 🔍 Dashboard Highlights

### Transaction Report
- Revenue breakdown by **Card Type** (Blue, Silver, Gold, Platinum)
- Revenue by **Education Level**, **Customer Job**, **Expenditure Type**
- Quarterly revenue and transaction count trends
- Chip vs Swipe vs Online usage analysis

### Customer Report
- Weekly revenue trends throughout **2023**
- Revenue segmented by **Age Group**, **Income Level**, **State**
- Customer job and dependent count analysis
- Gender-based performance comparison
